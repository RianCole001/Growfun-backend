## 🐛 Troubleshooting

### Still seeing 401 errors after login?
1. Open DevTools (F12)
2. Go to Application → Local Storage
3. Check for `user_access_token`
4. If missing, logout and login again

### Investment not appearing?
1. Refresh the page (F5)
2. Check amount is at least $30
3. Check you have sufficient balance
4. Check browser console for errors

### Admin credit not showing as deposit?
1. Refresh the deposits page
2. Check "All" or "Completed" filter
3. Look for payment_method: "admin_transfer"
